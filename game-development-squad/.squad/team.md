## Project Context

- **Project:** brutalgames.online — Retro Game Arcade
- **Description:** A retro game arcade site with a consistent neo-brutalist design theme. Each game ships as a single zero-dependency index.html file inside its own subfolder.
- **Stack:** Pure HTML, CSS, JavaScript — HTML5 Canvas — zero external dependencies
- **Design Theme:** Neo-brutalist — bold colors, thick borders, monospace type, harsh shadows, no gradients
- **Delivery Format:** One subfolder per game, each containing a single `index.html`
- **Rendering:** All graphics drawn procedurally via Canvas 2D API (no image assets)
